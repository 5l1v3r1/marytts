SETTINGS HELP

Warning: the settings of the modules are connected: If you change a setting in one module, 
then maybe the next module will not work anymore! 

The settings are set up so that everything works, so normally you will not want to change anything.
Settings that you might want to change and that do not interact with other settings, are marked with a star (*).


GlobalProperties:
db.basenameFile      : the file containing the list of files that are used to build the voice
*db.domain           : general or limited
*db.gender           : female or male
db.labDir            : the directory containing the label files. Will be created if it does not exist.
db.labExtension      : the extension of the label files, default: ".lab"
*db.locale     	     : de, en or en_US
*db.marybase      	 : the directory containing the local Mary installation
db.maryxmlDir    	 : the directory containing maryxml representations of the transcripts. Will be created
					   if it does not exist.
db.maryxmlExtension  : the extension of the maryxml files, default: ".xml"
*db.rootDir          : the directory in which all the files created during installation will be stored.
					   Will be created if it does not exist. 
*db.samplingrate     : the sampling rate of the wave files, default: "16000"
db.textDir           : the directory containing the transcript files. Will be created if it does not exist.
db.textExtension     : the extension of the transcript files, default: ".txt"
*db.voicename        : the name of the voice, one word, default: "cmu_us_bdl_arctic"
*db.wavDir           : the directory containing the wave files. If it does not exist, an Error is thrown.
*db.wavExtension     : the extension of the wave files, default: ".wav"


Properties for module praatPitchmarker:
*praatPitchmarker.command    : the command to start praat; default is "praat"
praatPitchmarker.corrPmDir   : the directory containing the corrected pitchmarkfiles. Per default the same
							   as the directory containing the pitchmarks. Will be created if 
							   it does not exist.
praatPitchmarker.corrPmExt   : the extension of the corrected pitchmark files, default: ".pm.corrected"
*praatPitchmarker.maxPitch   : maximum value for the pitch (in Hz). Default: female 500, male 300.
*praatPitchmarker.minPitch   : minimum value for the pitch (in Hz). Default: female 100, male 75.
praatPitchmarker.pmDir       : the directory containing the pitchmarks. Will be created if it does not exist.
praatPitchmarker.pmExt       : the extension of the pitchmark files, default: ".pm"
praatPitchmarker.pointPExt   : the extension of the point process files, default: ".PointProcess"
praatPitchmarker.tempScript  : the filename of the temporary script used by praat
